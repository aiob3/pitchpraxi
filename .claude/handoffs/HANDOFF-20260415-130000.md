## Decisions Made
1. **Engine**: whisper.cpp base + Intel MKL venceu (4 engines testados). Motivo: 6x mais rapido, 13x menos RAM que openai-whisper.
2. **Hotkey**: Alt+Backspace (pynput reporta como "backspace" no ABNT2). Motivo: Alt+A conflitava com WhatsApp.
3. **Paste**: clipboard + xdotool (Ctrl+Shift+V terminal, Ctrl+V GUI). xdotool type descartado (lento para texto longo).
4. **Config**: JSON em ~/.config/speechfire/config.json. Motivo: usuario pode editar sem recompilar.
5. **Skills portaveis**: Plugin no GitHub privado, nao em ~/.claude/skills/. Motivo: PC Windows perdeu skills locais.
6. **Regra fundamental**: NUNCA modificar codigo homologado — apenas ACRESCENTAR.
